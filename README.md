# TOXICITY

TOXICITY é um jogo 2D desenvolvido em Python com Pygame, onde o jogador controla um caminhão de lixo que percorre uma estrada coletando caçambas enquanto evita perigos pelo caminho.

O objetivo é coletar todas as caçambas necessárias antes de colidir com obstáculos como espinhos ou barris radioativos.

O jogo foi desenvolvido com foco em aprendizado de desenvolvimento de jogos com Python


# Gameplay

O jogador controla um caminhão de lixo que se move verticalmente pela estrada.

Durante a partida:

* Colete caçambas de lixo
* Evite barris radioativos
* Evite espinhos na estrada

Se o caminhão atingir um barril radioativo, ocorre uma explosão e o jogo termina.
Se o caminhão atingir um espinho, o pneu fura e o jogo termina.

Ao coletar todas as caçambas necessárias, o jogador vence a partida.

---

# Controles

| Tecla | Ação                       |
| ----- | -------------------------- |
| ↑     | Move o caminhão para cima  |
| ↓     | Move o caminhão para baixo |
| ENTER | Selecionar opção no menu   |
